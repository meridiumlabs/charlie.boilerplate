1 Buttons

Any element can be used as a button e.g. <a> and <button>. Make sure to use aria role attribute role="button" if <a> is used as actual buttons rather than anchors.

  • .button--secondary -

    Used by secondary actions e.g. cancel.

  • .button--small -

    Small button.

  • .button--cta -

    Call to action.

Lorem ipsum
.button--secondary Lorem ipsum
.button--small Lorem ipsum
.button--cta Lorem ipsum 
<a href="/" class="button {$modifiers}" role="button">Lorem ipsum</a>

1.1 Button with icon

You can use any icon with a button

<a href="/" class="button" role="button">
    <span class="icon icon-search" aria-hidden="true"></span>Lorem ipsum
</a>

    <span class="icon icon-search icon--lg" aria-hidden="true"></span>Lorem ipsum
